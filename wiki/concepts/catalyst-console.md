# Catalyst Console

## Definition

The Catalyst web console is the primary interface for managing [[zoho-catalyst]] projects, services, and components. It provides dashboards, metrics, settings, environment switching, notifications, and access to all developer tools. [Source: catalyst-general-topics.md]

## Key Aspects

- **Left menu**: Navigate between all services inside a project
- **Component grouping**: Components organized by functionality, features, and purpose
- **"Start Exploring" requirement**: Must click before a service can be deployed to production
- **Dashboards**: Serverless and Cloud Scale services have dedicated dashboards

### Dashboard Metrics
- Sync Functions Executions (today)
- Async Functions Executions (today)
- Total Component Calls (today)
- Cron Executions (today)

### Header Features
- Environment switcher (Development / Production)
- "Deploy to Production" button
- Notification center (cross-service events)
- User account info with Catalyst user ID
- Organizations drop-down for multi-org management
- SDK download links
